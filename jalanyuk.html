Jalanyuk

Di Project ini saya sebagai coder atau programmer tugas saya membuat menu navbar mencari tempat hosting , menginstall plugin - plugin yang dibutuhkan , Membuat menu - menu pada web tersebut. Website ini ditujukan untuk para traveler yang menyediakan tiket pesawat, Rental mobil. Pada website inipun para pengguna jasa kami dapat memberikan pengalaman / Review bagaimana hotel atau tour yang kami sediakan. Sehingga dapat membuat pengunjung lain mempercayai menggunakan jasa jalanyuk tersebut.

Pada Website ini saya menggunakan Boostrap , jquery , javascript , html , css dan Local web server yaitu XAMPP dengan phpmyadmin sebagai database awalnya. Pada website ini sudah tersedia fitur login dan register sehingga para pelanggan dapat menabung point pada akun masing - masing user setelah menggunakan jasa ini.

Di sini untuk login dan register saya membuat agar html dan php konek ke javascript sehingga dapat menambahkan beberapa validasi agar data yang dimasukan real dan tidak terjadi redudansi data. Jika user berhasil register maka akan menampilkan notifikasi di atas menggunakan alert("..") pada javascript.

Dapat dilihat dari Screen Shot di atas merupakan halaman home pada website ini. Lalu saya ingin website ini agar dapat diakses secara online dengan Link URL. Saya menyewa salah satu Penyedia web hosting di kota saya. Lalu memasukan project ini ke server mereka dan melakukan beberapa konfigurasi sehingga dapat diakses.
