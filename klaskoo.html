Klaskoo Mobile App

Pada halaman "Login". user diminta untuk memasukan username. Isi dari username tersebut akan dilembar ke API untuk mengecek apakah terdaftar didalam database. Jika terdaftar maka masuk halaman login yang usernamenya telah terisi otomatis. yang telah di setting oleh Front-end sehingga user tinggal memasukan password.

Jika username / email belum terdaftar maka user akan masuk ke halaman "Register". Lalu disini user juga dapat melihat apa saja yang dapat dilakukan pada aplikasi klaskoo dengan click sebuah Text 'explore klaskoo' maka akan dibawa ke halaman onboarding lalu ditampilkan beberapa slide yang dapat di geser.

Setelah masuk user akan diberi pilihan apakah akan masuk sebagai owner, staff atau parent. Setelah memilih owner maka akan ditampilkan nama - nama sekolah lalu pilih salah satu sekolah untuk masuk ke halaman dashboard. yang dimana terdapat sidebar untuk memilih berbagai macam menu atas fitur yang tersedia.

Pada gambar dibawah menunjukan menu sidebar. Menu list murid dan dapat beberapa filter untuk mencari nama murid yang spesifik dan adanya fitur class untuk mengecek apakah ada jadwal kelas ditanggal tersebut.

Mohon maaf tidak semua fitur atau menu dapat saya tunjukan karena adanya beberapa classified file company policy yang berlaku. sekian.
