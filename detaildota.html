Dota Marketplace

Pada bagian “Login”, User diminta untuk memasukan Username dan password yang sudah terdaftar di dalam database. Bagi calon user yang belum mendaftar, dapat langsung mendaftar melalui tulisan “Sign Up” yang akan membawa user ke page “Register” untuk mengisi data diri yang dibutuhkan system.

Pada bagian “Register” ini User diminta untuk mengisi beberapa informasi seperti Full Name, Username, Password, dll untuk user yang belum mendaftar. Jika user sudah mengisi bagian Register maka user akan dialihkan ke page “Login”

Setelah user melakukan Login, user akan langsung diarahkan ke menu Main Form dimana di menu ini user dapat melihat seperti nama user yang sudah didapatkan melalui username yang user sudah daftarkan dan jumlah balance yang user punya. Di page ini juga terdapat navigation bar yang berisi empat menu yaitu menu Top Up, History, Log Out, serta Database. di page ini juga user dapat memilih dari beberapa list item yang dijual serta stock dan harga yang tersedia. Jika user ingin membeli salah satu item dapat langsung menekan tombol “Buy” yang mana akan membawa user ke page Buy Form.

Jika dilihat dari page dibawah mengapa Error input item to itemtable karena item sebenarnya sudah masuk ke itemtable jadi dia Error. Jika pertama kali dia sukses tidak menampilkan Toast Error.

Pada saat user masuk ke dalam menu form, user akan mendapatkan warn berbentuk light box yang berisikan tentang “persetujuan user untuk mendapatkan SMS berupa pemberitahuan transaksi” user diharuskan untuk menyetujui hal itu untuk melakukan transaksi. Ketika user sudah menyetujui, maka user diminta untuk mengisi quantity item yang user butuhkan, quantity tidak boleh melebihi dari stok yang ada dan balance tidak boleh kurang dari jumlah harga yang tertera ketika menekan tombol total. ketika 2 hal itu sudah sesuai, maka user dapat langsung menekan tombol “BUY” dan akan langsung mendapatkan pesan berupa berhasil atau tidaknya transaksi.

Pada bagian “Top Up” ini, User dapat menambahkan balance dengan ketentuan lebih dari 50.000 dan memasukan Password yang sudah terdaftar pada database.

pada bagian “Seller Location” ini, User dapat melihat lokasi pada tiap item yang berbeda melalui tombol “Show” pada page Buy Form.
