Denver Watch

Sebelumnya untuk projek tersebut saya menggunakan software Microsoft SQL Server Management Studio. Pertama - tama saya membuat terlebih dahulu beberapa struktur table yang dibutuhkan kasus tersebut sesuai dengan syarat yang diperlukan. Terdapat pada file createDatabaseSystem.sql

Kemudian saya membuat / memasukan data untuk dimasukan kedalam table yang telah dibuat diatas. Sehingga data tersebut dapat diolah dengan baik. Pada saat simulasi transaksi dibawah ini saya sertakan beberapa contoh querynya.
Lalu disini juga saya membuat beberapa contoh transaksi dari dengan tabel dan data yang telah tersedia.
