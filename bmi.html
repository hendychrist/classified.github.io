BMI Calculator

Pertama - tama yang pasti mendesign UI lalu mengkoneksikan UI ke Controller. Na lalu disini bagaimana rumus agar dapat menghitung hasil dari BMI tersebut. Jadi dari project ini saya belajar import Foundation itu untuk apa, import UIKit itu untuk seperti UISlider , UILabel, UIViewController.

Selanjutnya saya menyeting Segue dari aplikasi tersebut. Segue adalah transisi antara 2 view controller pada storyboard file. Result kalkulasi jika bmiValue kurang dari 18.5 maka terlalu kurus dan berat badan kurang ideal maka akan print Eat more pies.

Jika hasil bmiValue diantara < 24.8 && > 18.5 maka berat badan anda sudah ideal.
Jika hasil bmiValue > 24.8 maka berat badan anda terlalu gemuk.
